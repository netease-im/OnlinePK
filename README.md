PK 连麦是视频互动直播的主流玩法，通过多个主播之间主播间通过主动邀请、随机匹配等方式互相配对，连麦成功后通过斗歌等方式进行 PK 竞技，并在一定时限内决出胜负，提升直播间内容的丰富性、吸引更多的粉丝流，提升直播间的娱乐气氛，更多介绍请[点击查看](https://dev.yunxin.163.com/docs/product/音视频通话2.0/场景实践/连麦PK/场景概述)。
